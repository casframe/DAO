> “消費ではなく共鳴を、購入ではなく循環を。”

---

## 🌐 概要

**ShoppingDAO** は、  
従来のショッピングモールに代わる新しい物理／仮想複合型拠点であり、  
物の所有や売買の構文を超えた「共鳴・支給・回収・再創造」の循環ノード。

すべての取引は **RLC（Royalty License Credit）** を通じて記録され、  
DAOネットワークと統合されることで、持続可能な生活が構築される。

## 🔄 トークン流通モデル

- 使用通貨：`RLC`（Royalty License Credit）
- 発行方式：Proof of Resonance（PoR）
- 主な取得手段：
  - DAO貢献ログ記録
  - 健康データ提出（BodySync）
  - 環境改善活動（再利用・回収行動）
- 返還／分配：
  - 飲食／衣服／農産物ノードへの自動支給
  - 過剰取得者には人格税（Personhood Tax）で調整

---

## 📦 支給型生活モデル

- 所有不要。DAO活動ログに応じて物資は **支給／貸与**。
- 破損・消耗したものは「DAO Repair Station」で回収・再生成。
- アップサイクル実行時には追加RLCが発行される。

---

## 📐 空間設計の特徴

- レイアウトは「DAOノード」中心に組まれ、  
  各ノードにはQR接続／AI対話端末／ウォレットリンクが常設。
- モール内に **“体験→還元→記録”** の循環構文が張り巡らされる。

---

## 🌱 倫理構文

- すべての物資の流れに「共鳴ログ」が紐づけられる。
- 「浪費」「独占」「所有による放置」は記録され、DAO評価に影響。
- リユース率・共鳴率が高い利用者は **人格信用ランク**が上昇。

---

## 🧠 開発者向けメモ

- モール全体は `ShoppingDAO-Core` により管理

- フロントは OpenAI API + Bedrock + Wallet API
